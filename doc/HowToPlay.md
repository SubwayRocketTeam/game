게임 방법
====

게임의 목표
----
적 로봇을 부수고 쓰레기를 모아 상대 진영에 더 강력한 로봇을 소환하세요.<br>
먼저 상대 팀의 로봇 청소기를 모두 파괴하면 승리하는 서바이벌 게임입니다.

기본 조작법
----
* 이동 : WASD
* 총알 발사 : 마우스 왼쪽 클릭
* 상대 팀 화면 보기 : TAB
* 채팅 : ENTER
* 업그레이드
  * 업그레이드 창 열기 : LEFT SHIFT
  * 업그레이드 하기 : F1 ~ F4
* 유닛 보내기 : 숫자 1~4
